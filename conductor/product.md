# Voice Stall - Product Guide

## Initial Concept
Voice Stall is a local Windows dictation application designed for efficiency, privacy, and high accuracy. It leverages `faster-whisper` for speech-to-text processing, offering a global hotkey system and automatic text pasting for a seamless user experience.

## Target Audience
- **Power Users / Developers**: Those who need quick, keyboard-driven dictation to improve productivity.
- **General Writers / Office Workers**: Users looking for a high-speed alternative to typing for long documents or emails.

## Core Values & Principles
- **Local Privacy First**: All audio processing and text transcription are performed entirely on the user's machine. No data is sent to external servers.
- **High Accuracy STT**: Prioritizing the best possible speech-to-text conversion using advanced `faster-whisper` models and customizable profiles.

## Key Features
- **Global Hotkey & Auto-Paste**: Instant dictation activation and text insertion directly into any active application.
- **Dictation History & Analytics**: Local logging of previous dictations and timing metrics to help users track performance and latency.
- **Performance Profiles**: Support for switching between fast, balanced, and accurate models to suit different hardware or needs.

## User Interface & Design
- **Deep Night Minimalist**: A premium, high-contrast dark theme with a frameless, floating window design and rounded corners.
- **Opalescent Content**: 100% opaque text and interface elements on a semi-transparent background for maximum legibility and style.
